# FunFit – Workout Journal & Personal Trainer App

**Problem:** Existing fitness apps lack flexibility for personalized workout tracking and scalable performance.

**Solution:** FunFit is a full-stack workout journal that allows users to track workouts, preferences, and progress using a flexible MongoDB-backed data model.

## Tech Stack
- TypeScript, React
- Node.js, Express
- MongoDB
- Webpack

## Key Features
- User profile & workout history tracking
- Optimized client–server communication
- Scalable backend architecture
